# MemoryAI — Feedback & Issues

This repository is the official place to report bugs and request features for **MemoryAI**, the 100% private AI assistant that runs entirely on your device.

## How to Submit

1. Check [existing issues](../../issues) to avoid duplicates
2. Click **New Issue**
3. Choose **Bug Report** or **Feature Request**
4. Fill in the template and submit

## Download MemoryAI

- [App Store](https://apps.apple.com/app/6758719586)
- [Google Play](https://play.google.com/store/apps/details?id=com.ts.memoryai)

## Privacy

MemoryAI runs 100% on-device. No data is sent to any server. See our [Privacy Policy](https://alessandrofacchini.github.io/memoryai-privacy/).
